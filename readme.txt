Ceneo Plugin wyświetla najpopularniejsze produkty z kategorii lub wybrane przez Ciebie.

== Description ==

Ceneo plugin – prosta wtyczka, która pozwala na wyświetlanie na stronie rankingu produktów z serwisu Ceneo.pl. 
Po zainstalowaniu pluginu mamy możliwość wyświetlania na stronie najpopularniejszych produktów z wybranej 
kategorii tematycznej z Ceneo bądź też każde inne dowolne zestawienie produktów z Ceneo.

Wtyczka jest rozwiązaniem polecanym, zarówno dla piszących o nowych technologiach, jak i dla blogerów piszących 
o modzie, urodzie czy wyposażeniu wnętrz. Pomoże wzbogacić kontent o rzetelne informacje z serwisu, 
który miesięcznie odwiedzany jest przez 5 milionów Polaków.

== Installation ==

Instalacja wtyczki jest bardzo prosta. Wystarczy załadować Ceneo plugin, aktywować i skonfigurować widget wedle upodobań. 

= Konfiguracja widgetu =

Konfiguracja polega na wyborze kształtu tabelki (pion czy poziom), wpisaniu nazwy (np. Najchętniej kupowane laptopy), 
wyborze kategorii produktów (np. Kremy do twarzy) bądź wyborze samych produktów. Prowadzący bloga decyduje także o tym, 
ile produktów znajdzie się w rankingu, a także o tym czy zastosować numerację produktów. 

Wprowadzając listę produktów podaj identyfikatory Ceneo (np. 8652742 -> http://www.ceneo.pl/8652742 ) 
rozdzielaj je przecinkami, np: 123456,345678,567890 .
Jeśli wprowadzisz zarówno listę produktów jak i nazwę kategorii, to wyświetlą się produkty z listy, 
ale pokaże się też nazwa i link do kategorii.
